Image Search Abstraction

Usage:

Search for images! Include your custom search string in the URL to receive a JSON response of results.

To search:

Follow this service's URL with /search.

  • For example: UPDATE https://desolate-inlet-99642.herokuapp.com/new/www.iamthecoolestwebsiteever.com/scooooobiedoooobiedooooooo

Include ?offset=# where # is the result number where you would like to start the returned listing.

  • For example: UPDATE https://desolate-inlet-99642.herokuapp.com/new/www.iamthecoolestwebsiteever.com/scooooobiedoooobiedooooooo
